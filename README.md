# Final-Group-Project
This was a group effort in order to create. We Wanted to create a website that would allow dog lovers to explore the many wonderful breed of dogs that there is to offer in the world.

The home page is suppose to be laid out in a simple fashion that should allow users to navigate our website no matter how profecient they are at using electronics. We wanted to go for a colorful yet simple color scheme that would allow our pictures and words on the screen to stand out.
The second page is the Dog List page in which we orginized the dogs in alphabetical order to allow users to scroll in order to find the desired dog they are looking for. We also made all the pictures interactive to where if you click on them additional information pops up about the dogs such as are they hypoalergenic and a description of the dog.
The third page is the Gallery. This is to showcase just amazing pictures that we found of dogs. This allows dog lovers to scroll a gallery of cute dogs without being worried about the extra info.
The last page is the More information page. We added this to allow users to sign up for a newsletter and to give additional information about local shelters in the tulsa area in case people are activly looking for dogs to add to thier family. We also have a link that allows users to add their email information.
